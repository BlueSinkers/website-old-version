Human Security Detector

Human Security Detector uses your webcam to monitor for unauthorized access. When a person is detected, it prompts for a password and, if not entered, triggers an alarm, takes a photo, and can initiate other configurable actions like shutting down the computer. It uses a trained-version of the YOLO Object Detection System.

Sparsity

An implementation of CSR (Compressed Sparse Row) representation and various matrix methods such as Matrix Multiplication, Cholesky Decomposition, and QR Decomposition. There is a near-finished implementation of a Collaborative-Filtering scheme using my implementation of CSR (a recommending scheme based on what other users "liked") and an implementation of Conjugate Gradient Descent, a deterministic approach to solving sprase DiffEqs.
