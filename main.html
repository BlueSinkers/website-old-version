Mathematics of Computer Vision

This project is an educational resource made for learning more about Computer Vision consisting of 60+ pages of TexEd up notes and various Python Demos (Documented Jupyter Notebooks). These notes were also presented in a slideshow format at various robotics events and conferences (including the Battle O Baltimore conference) and were a result of compilation of various resources (referenced throughout). Some relevant topics covered include the PnP Model, object detection, facial recognition, digital signals processing, image compression, and deep learning (CNNs, famous architectures, and basic math).

Human Security Detector

Human Security Detector uses your webcam to monitor for unauthorized access of your computer. When a person is detected, it prompts for a password and, if not entered, triggers an alarm, takes a photo, and can initiate other configurable actions like shutting down the computer. It uses a trained-version of the YOLO Object Detection System to detect humans, and TKinter for the window-screen (which is immovable/uncloseable until password-entry). The code is thoroughly documented and discusses YOLO's architecture issues such as installing it through DarkNET etc.

Sparsity

This project contains implementations of Compressed Sparse Row Matrix representation (CSR), various matrix methods (multiplication, QR/LU/Cholesky Decomposition, etc), a Collaborative-Filtering scheme using my implementation of CSR (a famous recommending scheme), and Conjugate Gradient Descent, a deterministic approach to solving sparse Differential Equations. It's implemented in C and has comparable run-times to similar functionalities of scipy.sparse, Python's standard module for sparse matrices. It is inspired by Dr. Bahar Asgari's PhD Thesis.
